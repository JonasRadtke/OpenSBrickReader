# OpenSBrickReader
OpenSBrickreader is an Android app for reading the new Lego Smart Brick NFC tags.  
The data read can be saved in a binary file.  
The binary file can be imported into a new tag. Tested with an ICODE SLI tag from NXP.   
The copied tag is recognized by the Smart Brick without any problems.

## Original NFC Tags
Lego uses ISO 15693 Type 5 NFC tags
- Manufacturer: EM Microelectronic
- Size: 256 bytes

## Tested NFC Tags
- Manufacturer: NXP
- Type: ICODE SLIX
- Size: 112 bytes

## Tested Smartphone
- Samsung GALAXY S20 FE 5G G781B
- amsung Galaxy S21 5G SM-G991B
